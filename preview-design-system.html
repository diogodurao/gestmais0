Design Checklist
Rule Check
Does this work without color? Text hierarchy should be clear in grayscale
Can I remove the card and keep clarity? If yes, use section + table instead
Is there more than one primary action? Only one btn-primary per screen
Am I using an icon for decoration? Remove it. Icons only for essential function
Would this look ridiculous printed? If yes, simplify
Am I using spacing other than 8/16/32? Stick to the locked scale
Am I using a font size not in the scale? Use only: 9, 10, 11, 12, 13, 14, 16, 20px
